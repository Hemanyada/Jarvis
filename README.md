🤖 Jarvis – AI Smartphone Comparison Assistant

Jarvis is an AI-powered web application that helps users compare smartphone prices across multiple platforms, analyze specifications, and get intelligent recommendations using AI.

🚀 Features
🔍 Price Comparison
Fetches prices from:
Amazon
Flipkart
Croma
Official Brand Stores
Highlights the best deal
Provides direct buy links
📊 Price Visualization
Generates a bar chart for price comparison across platforms
Helps users quickly identify the cheapest option
📑 Specifications
Displays key smartphone specs:
Battery
Display
Processor
Camera
Data fetched using EXA AI
⚖️ Compare Smartphones
Compare multiple phones side-by-side
Compares:
Battery
Display
Processor
Camera
Tags
🤖 AI Assistant
Ask queries like:
“Best phone under ₹30,000”
“Is this phone worth buying?”
Powered by Gemini API

⚙️ Tech Stack
Frontend: Streamlit
AI & APIs:
EXA AI (Specifications)
Gemini API (AI Assistant)
Language: Python
🔄 Workflow
User enters smartphone name
System fetches prices from multiple platforms
Specifications are fetched using EXA AI
Data is processed and compared
Results displayed with charts and tables
AI assistant handles user queries
